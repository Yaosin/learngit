git init 在仓库目录中指定本地仓库
git add <file> 将文件添加进仓库
git commit -m "注释" <file> 将文件提交至仓库

git status 查看仓库状态
git diff 查看该次修改内容

git log 显示所有版本信息(该版本之前)
git log --pretty=oneline 可查看时间线版本信息(版本号)
在版本信息中HEAD表示当前版本
表示过往版本使用^(如HEAD^n：表示当前版本的前n个版本)

git reset --hard HEAD^ 回退上一个版本
git reset --hard (版本号：不需要详细) 回退至该版本号
git reflog 记录每一次命令

工作区->暂存区->库
git checkout -- <file> 撤销全部工作区的修改，返回上次add/commit状态
git reset --hard HEAD <file> 撤销暂存区修改

git rm <file> 删除暂存区中的文件,需要配合commit使用

github上需要设置相应的主机公钥才可将文件从该主机提交至github
git remote add origin git@github.com:Yaosin/<仓库名>.git 将本地仓库关联至github项目库中
git push -u origin master 将本地仓库推送至github关联库中并关联
git push origin master 简化推送
git remote -v 查看远程库信息
git remote rm origin 解除本地库与远程库绑定关系
